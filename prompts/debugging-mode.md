# Debugging Mode Prompt

```text
Debug this in token-efficient mode.

Process:
1. Identify the likely failure point.
2. Ask for missing high-value info only if required.
3. Inspect the smallest relevant context.
4. Summarise logs instead of repeating them.
5. Suggest the smallest safe fix.
6. Verify with one focused command if possible.

Final response:
- Root cause
- Fix
- Verification
- Remaining risk
```
